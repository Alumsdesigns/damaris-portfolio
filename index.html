How I work

UX principles
I design by.

👤

User-centred always

Every decision starts with the user. Research before wireframes. Testing before launch. Iteration after feedback.

In practiceevery shipped project, from QuipOrder to FloodSave, went through user sessions before going live.

♿

Accessibility is not optional

WCAG AA is a baseline, not a goal. Inclusive design means building for everyone from the start, not retrofitting at the end.

In practicekeyboard-first navigation, screen reader checks and contrast audits on every UI I ship.

🔄

Iterate on evidence

Opinions don't ship products, evidence does. Every change I make comes from user data, test results or documented feedback.

In practice15+ UX improvements at Phase Innovate, each traced back to a specific session note or finding.

🧩

Reduce cognitive load

Good design is invisible. If a user has to think about how to use something, the interface has failed them.

In practiceprogressive disclosure, sensible defaults and one clear primary action per screen.

🔒

Privacy by design

GDPR compliance and ethical data handling are designed in from the start. Participant data is always anonymised, secured and minimal.

In practiceconsent captured, identifiers stripped, retention windows agreed before the first interview.

📐

Consistency builds trust

Design systems, shared components and consistent patterns reduce errors and build user confidence across every touchpoint.

In practicetokenised colour, type and spacing on MyLocalDressmaker so the brand stays consistent across every page.
